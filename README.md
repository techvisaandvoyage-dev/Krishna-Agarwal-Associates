# Krishna Agarwal Associates

This is a small static website containing `index.html` and `privacy-policy.html`.

To publish the site with GitHub Pages:

- Open the repository on GitHub
- Go to Settings → Pages
- Under "Source" choose the `main` branch and `/ (root)` folder
- Save — the site will be published at `https://<your-username>.github.io/<repo-name>/` or the custom domain if configured.

Alternatively, use the GitHub CLI to configure Pages if available.
